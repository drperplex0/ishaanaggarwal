# Website Repository

This repository contains the HTML, CSS, and JavaScript files for the personal website of Ishaan Aggarwal. Each webpage is designed to present different facets of my online portfolio, including my projects, reading list, blog, about page, and contact information.

## File Structure

- `index.html` - The main landing page of the website.
- `projects.html` - A showcase of my projects.
- `reading-list.html` - A compilation of books and articles that I recommend.
- `blog.html` - My personal blog where I share insights and stories.
- `about.html` - Information about my background and experiences.
- `contact.html` - A page for visitors to reach out to me.
- `style.css` - The stylesheet that contains all the styling for the website.
- `scripts.js` - The JavaScript file for dynamic behavior on the projects page.

## Usage

You can clone it if you want a similiar template, my inspiration was from https://thanhvtran.com/
